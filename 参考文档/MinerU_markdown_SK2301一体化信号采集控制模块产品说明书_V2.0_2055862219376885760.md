# 一、产品概述

SK2301 是一款专为设备健康状态在线监测与预测性维护而设计的一体化信号采集控制模块，可自动、准确、高速、可靠地直接采集安装在设备上的振动、温度、转速、压力、

![image](https://cdn-mineru.openxlab.org.cn/result/2026-05-17/125a25f2-2bf5-4759-841b-8cbbbe3f3844/7f4b50d28c02e00f822b6526ce78bf9a6e718e54f098d1295761b44b518fb418.jpg)


位移、开关等各种传感器输出的模拟信号，无需外加任何信号调理电路或者转接模块，并可以把现场采集到的数据实时自动上传到云平台。该产品广泛应用于旋转和往复运动机器，例如：风机、离心机、泵、电动机、减速机、工业风扇、热交换器/ 冷却塔、空冷岛风机、发电机、活塞压气机、离心式压缩机、碎石机/磨煤机、压缩机、泵、磨煤机等设备的在线健康监测。

# ※ 产品特点：

 SK2301 既可以按照采集计划把采集数据实时自动直接上传到云平台，也可以现场一台SK2301 或者多台 SK2301 布置一个我们公司的边缘计算智能网关，然后由智能网关统一数据管理运算之后，再上传到云平台。

 SK2301 无需外加各种调理转换模块就可以直接接振动、温度（PT100）、转速、压力、位移、开关等各种传感器输出的模拟信号，覆盖设备状态监测各种常用参量对应的传感器。

 我们可以提供从各种传感器->监控模块->边缘计算智能网关->定制特定的计算处理软件等一揽子系统化的产品和服务。

 多台 SK2301 通过 10/100M 自适应以太网连接到交换机，组建一个局域网，让信号采集点扩展足够多，传输距离足够远。

 面板无任何操作旋钮开关，所有对采集仪的操作通过软件远程控制，外观小巧，且功能完整。

每个产品都具备独立可设的 IP 地址、子网掩码、网关，通过修改相关参数，可直接插接在任何一个现有的局域网内，设置后的参数直接保存在仪器内，掉电不丢失。

 可配置 TF 卡功能，具备离线采集功能，并且方便下载数据。

 匹配 WIN XP/WIN 7 操作系统下信号采集与控制测试软件。

# 五、使用方法

1、产品进行联机操作时，首先确保进行连接测试的计算机 IP 地址、网关、子网掩码的设置与该产品默认的网络参数在同一个局域网内。该产品默认的网络参数为 IP：192、168、1、198，端口 1600，

网关 192.168.1.1，子网掩码 255.255.255.0。

2、按照面板标识说明所述，接好传感器的信号、电源、以太网线。

3、预热十分钟再进行测量。

4、打开配套软件，选择数据采集->以太网产品测试->FKPro测试程序。

5、依次点击“连接仪器”、“读仪器信息”、“采集设置”按钮。

6、可参考提供的产品基本操作指南文档设置同步采集参数、采集模式等参数，并点采集设置按钮设置。如果勾选上“波形显示”页面里的“保存采集数据”，可以采集的同时保存采集数据

7、点“开始连续采集”按钮开始采集。在“波形显示”页面可以同时显示高速采集通道中任意 4 路的波形，右边“通道显示选择”可以选择 4 路波形中每路波形对应的通道序号，以及是否显示波形，并且在右边的文本框里实时显示通道当前所有窗口波形数据的平均值。

8、点“停止连续采集”按钮停止采集。

以上只是最常用的实时高速采集功能使用说明，如果需要进行更复杂的功能测试和验证，以及编程等相关工作，请与本公司技术支持联系。

# 六、参数修改与恢复

目前该产品掉电不丢失的参数有 IP、子网掩码、网关、看门狗时间、采集参数等，下面主要以产品的IP、子网掩码、网关为出厂默认参数为例，说明修改以太网参数设置及恢复过程。该产品默认 IP：192.168.1.198，子网掩码：255.255.255.0，网关：192.168.1.1。

# 1.修改以太网参数步骤

1）按照参数说明和面板标识说明连接好合适的电源；

2）用能正常上网的网线把产品与计算机的网口连接起来，此时产品网口的两个指示灯必须都亮；

3）把计算机当前的 IP、子网掩码、网关记下来，备用；

4）设置计算机的 IP 为 192.168.1.XXX，XXX 只要不为 1、198、255 即可，子网掩码为 255.255.255.0，网关为 192.168.1.1；（这步目的就是暂时让计算机与产品在同一个局域网内，以便能相互通讯）

5）打开测试程序的“网络参数”页面；

6）填写产品当前的 IP到“当前 IP地址：”对话框，如果无法确认当前 IP地址，就先恢复出厂参数再修改该产品的以太网参数；

7）填写需要设置的 IP、子网掩码、网关到“IP 地址：”、“子网掩码”、“网关”三个对话框；

8）确认无误后，点“设置”按钮，如果弹出“修改成功！”对话框，表示修改成功，无需给产品重

新上电，产品已自动初始化为设置后的参数，以后访问该产品都要用修改后的参数访问。如果未设置成功，可通过对产品做恢复出厂参数操作、检查网线是否正常、检查操作步骤是否有问题、计算机是否两个网卡（禁止掉一个网卡再试），几个方面去排查问题。

9）修改成功后的参数掉电不丢失，该产品可直接连接到相适应的局域网内，通过局域网内任何一台计算机访问使用；

10）把第 3）步记下来的计算机的 IP、子网掩码、网关恢复，把网线连接复原。

# 2.恢复出厂参数步骤

在上电的情况下，用针状硬物插入网口右边的小圆孔，轻轻按下内部的弹性按钮，按 5-8 秒左右再松开，此时产品 IP、子网掩码、网关便已经恢复为出厂默认参数。

# 七、软件开发

本公司提供的采集测试软件只供产品功能测试，用户需要根据自己的特定应用再编写上位机程序进行二次开发，当然也可以购买我们现有正式软件，或委托我们进行上位机软件定制开发。

本产品采用以太网通讯，需要按照我们公司的 FkPro 协议，基于 socket 编程进行连接、设置、读数等相关操作，对操作系统和编程语言没有限制，只要求能建立 socket连接。

采用 FkPro 协议编程时，所有指令格式、寄存器地址、字段介绍参照《FKPro 通讯协议及编程说明书》，和《FKPRO 指令举例说明》。编程之前，用户可先采用网上下载的《以太网调试助手》这个小软件，进行指令发送接收熟悉，然后再把指令嵌入到自己的程序里。
